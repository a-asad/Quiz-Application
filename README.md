# QUIZZER
A quiz app with questions from a wide variety of categories.

## FEATURES
- Choose number of questions you want
- Choose from 3 difficulty levels
- Choose questions from a large list of domains

## TECH STACK
- React Js
- Typescript
- Node js
- HTML
- CSS
